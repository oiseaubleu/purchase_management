# README

## 開発言語

- Ruby 3.3.0
- Rails 6.1.7.8

## 実行手順

```
$ git clone　git@github.com:oiseaubleu/wammande.git
$ cd wammande
$ bundle install
$ yarn add @babel/plugin-proposal-private-methods
$ yarn add @babel/plugin-proposal-private-property-in-object
$ rails db:create && rails db:migrate
$ rails s
```
